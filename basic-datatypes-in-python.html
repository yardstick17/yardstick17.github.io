Basic Datatypes in Python


Every variable has a type in Python. As everything in Python is an object. Datatypes are the classes, and the variables are instances of these classes.

type is used to determine the class an object belongs to and isinstance is used to check if an object belongs to a particular class.

>>> x = 4
>>> type(x)
<type 'int'>
>>> isinstance(x, str)
False
>>> isinstance(x, int)
True

Among many various data types in Python, these are the important types are listed below. - integer, floats: 3, 4.5 - string: 'hello world' - tuple: (4, 5) - list: [4, 're', 1.4] - set: {2, 4} - dictionary: {1: 'one', 2: 'two'}

  • Integers, Floats Python interprets the decimal digits without any prefix. For example: 
>>> x = 4
>>> x
4

For binary numbers append 0b or 0B before the number representation.

>>> x = 0b111
>>> x
7

For octal numbers append 0o or 0O before the number representation.

>>> x = 0o101
>>> x
65

For hexadecimal numbers append 0x or 0X before the number representation.

>>> x = 0x101
>>> x
257

Floating points are accurate upto 15 decimal places. In representation, numbers are separated by a dot(.).

>>> x = 1.12345678910111213
>>> x
1.1234567891011122
